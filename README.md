# 🌊 Professional Minimalist Image Slider

A high-performance, fully responsive, and buttery-smooth image slider built with **Pure HTML, CSS, and Vanilla JavaScript**. No external libraries, no jQuery—just clean, optimized code.

---

## ✨ Key Features

* 🎯 **Fully Responsive:** Works perfectly on Desktop, Tablet, and Mobile.
* 🚀 **High Performance:** Zero dependencies, keeping your project lightweight.
* 🎨 **Modern UI:** Features Glassmorphism controls and a sleek "Pill-style" pagination.
* 🔄 **Smart Auto-play:** Automatically transitions slides with a "Pause-on-Hover" feature.
* ⌨️ **Keyboard Support:** Navigate through images using Arrow Keys.
* 🎞️ **Smooth Transitions:** Custom `cubic-bezier` animations for a premium feel.

---

## 🛠️ Built With

* **HTML5:** Semantic structure.
* **CSS3:** Flexbox layout, Glassmorphism, and Animations.
* **JavaScript:** ES6+ Vanilla JS for slider logic.
